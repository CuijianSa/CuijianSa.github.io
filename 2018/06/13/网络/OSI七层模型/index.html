OSI网络模型

OSI网络模型

OSI七层网络模型 TCP/IP四层模型 功能 协议 数据单位
应用层 应用层 面向用户应用，为用户提供对各种网络资源的方便访问服务 TFTP、HTTP、SNMP 消息
表示层 - 对来自应用层的命令和数据进行解析，对各种语法赋予相应的含义，并按照一定的格式传送给会话层 Telnet、SNMP 消息
会话层 - 向两个实体提供建立和使用连接的方法 SMTP、DNS 消息
传输层 传输层 向用户提供可靠的端到端差错和流量控制，保证报文的正确传输 TCP UDP 段segment / 数据包Packet
网络层 网络层 控制数据链路层和传输层之间的信息转发、建立、维持和中指网络连接 IP 、ICMP、ARP、RARP 包packet
数据链路层 数据链路层 负责建立和管理节点间链路 Ethernet、Arpanet、PPP 帧Frame
物理层 - 利用传输介质为数据链路层提供物理连接，实现比特流的透明传输 IEEE 802.A, IEEE 802.2 - IEEE 802.11 比特流Bit
